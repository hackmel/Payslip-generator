Payslip Generator

Instructions

  • Create an input csv file that contains the employee details with the following format.
  • First name, Last name, Annual salary, Super rate (%), Payment start date (MM-dd-YYYY)
  • One file can contain multiple employees.
  • By clicking the choose file button, browse and select the input csv file.
  • Click the upload button to upload the file. A succes message will appear once upload is complete.
  • Click download button to generate the payslip. A csv file will be downloaded into the local computer.

Sample format

Rommel,Suarez,60050,9%,11-08-2010

Guen,Yu,120000,10%,10-08-2010

Output format

"Name","Pay Period","Gross Income","Income Tax","Net Income","Super"

"Rommel Suarez","01 November- 30 November","5004","922","4082","450"

"Guen Yu","01 October- 31 October","10000","2696","7304","1000"
